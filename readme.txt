Allgemeine Regeln (damit das kein Chaos wird )

-Bitte zu jeder Aenderung oder Neuerung ein Kommentar hinzufügen, damit eine ausfuehrliche Dokumentation vorhanden ist.
-Beim Kommentar Datum und Autor einfügen 
-Entscheidungen und Aenderungen sind nicht nur im Code zu kommentieren, es sollte auch eine Dokumentation gepflegt werden
(Hier ist Disziplin und Teamwork gefragt :) )
-Entscheidungen hinterfragen und theoretisch zu untermauen ist kein Verbrechen
-Versionskontrolle gibt es nicht ohne Grund, also immer fein nachpflegen
-Wer sich mit Git noch nicht so auskennt, findet sicherlich genug Input in den Unweiten des Internets ;)
